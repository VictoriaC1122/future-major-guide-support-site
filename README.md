# 我決定自己的未來-大學選系指南 Support Site

Static GitHub Pages site for App Store support, privacy policy, product introduction, and user contact.

## Public URLs

- Support URL: `https://victoriac1122.github.io/future-major-guide-support-site/`
- Privacy Policy URL: `https://victoriac1122.github.io/future-major-guide-support-site/privacy.html`

## Site Goals

- Present the app as a polished education decision-support product.
- Provide App Store review friendly support and privacy information.
- Explain data limitations clearly without sounding alarming or promotional.
- Keep the page readable on mobile, tablet, and desktop.

## Structure

- `index.html`: public support and product introduction page.
- `privacy.html`: App Store privacy policy page.
- `styles.css`: shared responsive visual system.
- `assets/`: app icon, favicon, and Apple touch icon.

## Deployment Notes

The site is deployed from the `main` branch through GitHub Pages. After pushing, GitHub Pages and CDN cache can take a few minutes to show the newest HTML/CSS/assets.
